# R\_nuggets

Short R scripts and functions I have found useful.

This repo contains simple R scripts and functions that I have used repeatedly in teaching and/or research over the years. Some I wrote from scratch, some are modifications of other’s work, credited as relevant. There is a bias towards {base} R in these scripts and functions, as the {tidyverse} didn’t exist when I started my R journey. 

I created this repo as a place to start organizing my own scripts, and made it public in case others might find it useful. Please note that I am an anthropologist, not a programmer, so it goes without saying that there are probably more elegant ways to do any of the things I have included here. Sometimes my clunkiness is for pedagocial clarity, other times it reflects my limitations.
